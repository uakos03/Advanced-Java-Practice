&#x20;Ez a repo egy strukturált fejlesztési folyamatot dokumentál, amelynek célja a stabil, ipari szintű Java backend tudás kialakítása. A projekt a haladó objektumorientált (OOP) elvekre épít, és szisztematikusan készíti elő a terepet a modern, rétegezett szoftverarchitektúrák és a Spring Boot keretrendszer integrációjához.





\## Architektúra és Modulok





A repo monolitikus felépítést követ. Minden modul egy meghatározott technológiai koncepció gyakorlati elsajátítását követi.





\* \*\*`01-Collections-and-Generics/`\*\*: Dinamikus memóriakezelés és modern adatstruktúrák (`List`, `Set`, `Map`).



\* \*\*`02-Functional-Programming/`\*\*: Lambda kifejezések és a Java Stream API integrációja. 



\* \*\*`03-Data-Persistence/`\*\*: Relációs adatbázisok (SQL) kezelése Java környezetből (JDBC). 



\* \*\*`04-Layered-Architecture-Project/`\*\*: Záró szintetizáló projekt. Egy komplex, N-rétegű (Entity, Repository, Service) parancssori alkalmazás (CLI) felépítése, amely integrálja az üzleti logikát és a perzisztens adattárolást.





\## Technológiai Stack





\* \*\*Nyelv:\*\* Java SE (Core, OOP, Stream API)



\* \*\*Adatbázis:\*\* SQL / JDBC



\* \*\*Build Tool / Verziókövetés:\*\* Git, GitHub (későbbiekben Maven)



\* \*\*Környezet:\*\* IntelliJ IDEA
